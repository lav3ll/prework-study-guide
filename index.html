HTML

  • The head element contains information about the webpage
  • The body element represents the visible content shown to the user
  • The title element defines the documents title
  • The meta element contains metadata about the HTML document like charset, http-equiv & viewport

CSS

  • A margin indicates how much space we want around the outside of an element.
  • A padding indicates how much space we want around the content inside an element.
  • The border property assigns a border with a width of 5px that is solid and gray.
  • The box-shadow property adds shadow effects around an element's frame.

Git

  • git status: checks what branch we are currently on
  • git checkout -b branch-name: creates a new branch and switches to it
  • code . opens vscode in the current folder open in the command/gitbash
  • cd to change directory
  • checout lets you navigate between branches
  • git add -A: adds all files to staging area
